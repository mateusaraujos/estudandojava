Pasta > Git Bash Here

git clone https://github.com/mateusaraujos/estudandojava.git -> Clona o repositório.

cd estudandojava/ -> Entra no repositório clonado.

git status -> Verifica se está tudo certo. Vai mostrar se tiver alguma pasta nova no repositório local.

git add . -> Adiciona todos os arquivos ao controle de versão local. Falta o COMMIT.

git commit -m "Mensagem do commit aqui." -> Faz o commit local. Falta o PUSH para enviar o GitHub.

git push origin main -> Envia todos os commits locais para a nuvem.
